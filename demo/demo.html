Conway Life Canvas

A simple typescript module for animating HTML5 canvases with Conway's "Game of Life".

Usage

Create any number of html <canvas> element, and then call attachToCanvases with a selector that targets those canvases. See the source code to get a sense of what options are available.

Caveats

If you don't specify the width and height in the options to attachToCanvases the world will be generated based on the initial canvas size. If the canvas gets sized dynamically you probably want to specify a size that will be larger that you ever expect to see to avoid having to repopulate. Alternatively, the LifeCanvas can be resized and repopulated manually.
